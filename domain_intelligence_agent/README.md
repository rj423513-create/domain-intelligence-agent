# 🚀 Domain Intelligence Agent

Professional bulk domain analysis tool (WHOIS + DNS + Tech Stack).

## Features
- No API key required (uses free fallbacks)
- Streamlit UI for easy use
- Excel report with all details
- Handles 50-100+ domains

## Quick Start
1. `pip install -r requirements.txt`
2. `streamlit run agent/app.py`
